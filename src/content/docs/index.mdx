## 关于我
93年出生，2018年来到杭州，2022年已定居杭州良渚，之前是 UI 设计师兼产品助理，22年在现公司内部转岗为产品经理，因为工作的原因零零散散学了一些别的技能，
比如公众号运营、视频剪辑、平面设计、物料设计、甚至3D建模等。目前正在努力学习前端知识，因为自己时不时会有一些想法想要去实现。\
\
目前跟朋友已经做出的产品有：“我的 G 表”、“~今天吃什么~(已经停止运营)”，其中“我的 G 表”是面向 G-shock 用户推出的查询手表生产日期和生产地的简单的微信小程序，
目前已经积累了 3000+ 用户，获得 **5.0分** 的好评。
